# Chat App using Node.js & Socket.IO

A simple real-time chat application built using **Node.js**, **Express**, and **Socket.IO**.  
This project is created for learning and will be extended with more features like **chat rooms**, **usernames**, and more.

---

## Features (Current)

- Real-time messaging using Socket.IO
- Multiple users can chat simultaneously
- Messages broadcast to all connected clients
- Simple frontend using HTML & JavaScript

---

## Planned Features

- Chat rooms
- Usernames
- Join / leave notifications
- Better UI
- Message timestamps

---

## Tech Stack

- Node.js
- Express.js
- Socket.IO
- HTML, JavaScript
